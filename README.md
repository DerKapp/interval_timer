# Interval Timer App

Tabata timer for fitness

## Getting Started

1. Install dependencies
2. Run app

## Trouble Shooting

When ios build is not working, try this:

```
   flutter clean
   rm -Rf ios/Pods
   rm -Rf ios/.symlinks
   rm -Rf ios/Flutter/Flutter.framework
   rm -Rf ios/Flutter/Flutter.podspec
```
